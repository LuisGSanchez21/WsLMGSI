Definiciones de software, hardware y firmware

Software
refiere a los programas, aplicaciones y datos informáticos que se utilizan en una computadora o dispositivo electrónico para realizar diversas tareas y funciones.
Hardware
se refiere a todos los componentes físicos y tangibles de una computadora o dispositivo electrónico.
Firmware
es un tipo de software que se encuentra almacenado en la memoria no volátil de dispositivos electrónicos, como computadoras, teléfonos inteligentes, impresoras, enrutadores, cámaras digitales y otros dispositivos similares.
